Requirements (Code implementation):

- Once user will drag the table icon from the icon bar and drop into the 
  work area, it should ask for basic things like 

    `number of rows`
    `number of columns`

- Finally, based on the entries dynamically create a beautiful table whose
  tables & cells can be resizable. Cells can be editable.


Once we implemented the above functionality, we will go further to add extra
feature as mentioned below.

[Note: If user chooses CSV upload option then do not ask for rows/columns]

- Drag event should also ask for CSV file upload (So user is free to choose)
  any of the option.

- Once the upload will occur, get the path of CSV and path the PATH to Node
  API and get an array of values as 2D array (An array of arrays).

- In case of improper csv file specification, you will get `null` as return 
  value.

- Finally, the UI developer will either generate dynamic table based on 
  returned 2d array or again make an API call to get the related HTML code
  for table.

- It's better to use your own logic if you need extra customizations and 
  styles.
