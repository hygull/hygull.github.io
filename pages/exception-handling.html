 Python - Exception Handling


Exception handling is very important in programming. It allows an executing program to not terminate immedialtely after an exception occurs at any point.Let us see how to handle exception in Python. As I have not decribed the whole concept here that is why I have commented the lines. You can also visit Errors and Execptions and read it more clearly.



  """
      created_at       : 09 Feb 2017.
      updated_at       : 09 Feb 2017.
      aim of script    : To understand the concept of exception handling in Python.
      coded by         : Rishikesh Agrawani.
      Python version   : 2.7.12
  """
  a = 3
  b = 4
  name = 'Randy'

  try:
      print a+b           #It will execute
  except:
      print "Error while adding 2 numbers"

  try:
      print a+name        #It raises TypeError 
  except:
      print "Error while adding 2 numbers"     #It will execute


  cities = ["Raipur", 'Jagdalpur', 'Kondagaon', 'Dhamtari', "Nagri"]
  try:
      for index in range(len(cities)):
          print cities[index]                                   #It will execute to print all the cities
  except:
      print "Operation successful"


  try:
      for index in range(len(cities)+1):
          print cities[index]
  except:
      print "Error while accessing the out of range index"      #It will execute


  try:
      print 5+"Robinson"
  except:
      print "Can't add integer and string"            #It will execute
  finally:
      print "finally block execution is fixed"        #It will definitely execute

Let us look at a different flavour of handling exceptions. After that we will move towards the advance exception handling examples.Here we wll handle the exact Exception types. Let us see it in the following examples.

  
 #Example 1 - ZeroDivisionError

  try:
      a = 10
      c = 0
      print a/c                                               #This statement will raise ZeroDivisionError
  except ZeroDivisionError:                                   #This block will catch the Exception
      print "You are trying to divide something by 0 (zero)"  #So it will execute

 #Example 5 - IndexError


  try:
      fruits = ["Apple", "Orange", "Grape", "Blackberry", "Gooseberry", "Strawberry"]   #valid indexes are 0,1,2,3,4,5
      print fruits[4], fruits[5]                              #Gooseberry Strawberry
      print fruits[6]                                         #It will raise IndexError
  except IndexError as index_err:
      print "You are trying to access invalid index"
      print index_err                                         #list index out of range
