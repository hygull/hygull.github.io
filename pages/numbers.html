 Python - Numbers


Numbers are an important part of any programming language.
Numbers are immutable.
Numbers have been catagorized based on their representation on Number line.
    eg. positive, negative, boolean, complex, whole , even, odd,real etc.
Python supports 4 different numeric types. int, long, float, complex.
    Let's see the use of these numbers in Python.

 #Interactive mode programming - #3 (Number type conversion)

  >>> num1 = 2
  >>> num2 = 3L
  >>> num3 = 4.5
  >>> 
  >>> print num1
  2
  >>> print num2
  3
  >>> print num3
  4.5
  >>> 
  >>> print int(num3)           #conversion into int
  4
  >>> print long(num3)          #conversion into long
  4
  >>> l=long(num3)
  >>> l
  4L
  >>> print float(num1)         #conversion  into float
  2.0
  >>> 
  >>> print complex(4)          #conversion into complex
  (4+0j)
  >>> 
  >>> print complex(4,5)
  (4+5j)
  >>> 
  >>> print complex(num1,num3)
  (2+4.5j)
