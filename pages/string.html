 Python - string


string is defined a sequence of characters enclosed within single or double quotes.
Python has no support for character types like C/C++, C#, Java etc.
So we need to use a single character string for defining any character. 



  """

  """
  
 #Interactive mode programming - #1

  >>> name = "Julia Kelly"      #with double quotes
  >>> city = 'Bangalore'        #with single quotes
  >>> 
  >>> type(name)                #printing type
  <type 'str'>
  >>> 
  >>> type(city)                #printing type
  <type 'str'>
  >>> 
  >>> print name,city           #printing name & city
  Julia Kelly Bangalore
  >>> 
  >>> print name                #printing name
  Julia Kelly
  >>> 
  >>> print city                #printing city
  Bangalore
  >>> 
  >>> print "%s -- %s"%(name,city)          #1st way of formatting string variables
  Julia Kelly -- Bangalore
  >>> 
  >>> print "{} == {}".format(name,city)    #2nd way of formatting string variables
  Julia Kelly == Bangalore
  >>>

 #Interactive mode programming - #3 (indexing & slicing)

  >>> s = "ABCDEFGHIJ"
  >>> print s[0]            #Accessing characters using index
  A
  >>> print s[len(s)-1]     #Accesssing last character of a string
  J
  >>> id(s)             
  4558293776
  >>> 
  >>> s2 = s
  >>> id(s2)
  4558293776
  >>> 
  >>> print s2
  ABCDEFGHIJ
  >>> 
  >>> s3 = s[:]             #getting a slice having all the characters of string
  >>> id(s3)
  4558293776
  >>>
  >>> print s
  ABCDEFGHIJ
  >>> 
  >>> print s[:2]           #getting a substing having characters from start to 2nd character
  AB
  >>> print s[:3]           #getting a substing having characters from start to 3rd character
  ABC
  >>> print s[:len(s)]      #getting a substing having characters from start to end
  ABCDEFGHIJ
  >>> 
  >>> print s[1:len(s)]     #getting a substing having characters at index 1 to end
  BCDEFGHIJ
  >>> 
  >>> print s[1:]           #Same as above
  BCDEFGHIJ
  >>> print s[3:]           #getting a substing having characters at index 3 to end
  DEFGHIJ
  >>>
