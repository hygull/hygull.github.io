 Python - dictionary


Dictionary is defined as a key-value based data structure where each key maps to some value.

It is used for defining a data structure where we want to organize/group data by creating a meaningful relationship between keys( eg. name, age, branch) and their values( eg. 'Robert Prince', 22, "Mechanical engineering" ).

Interactive mode programming is a best way to start or have some basic knowledge of Python concepts. So here I'm starting with that.

A simple task for you beginners » Write a pyscript to print a list of all the keys of a dictionary including the nested ones.
The following examples will clarify your concepts.
I've used long variable names for making it undestandable for you.

 #Example - 1

  employee = {"name" : "Jordon Smith", "age" : 24, "monthly_salary": 200000.67}
  print employee["name"]              #Jordon Smith
  print employee["age"]               #24
  print employee["monthly_salary"]    #200000.67

  print employee                      #{'age': 24, 'name': 'Jordon Smith', 'monthly_salary': 200000.67}
  print employee.values()             #[24, 'Jordon Smith', 200000.67]
  print employee.keys()               #['age', 'name', 'monthly_salary']

  print employee
  


 #Example - 2

  student_details = {"name" : "Robert James", "age" : 55, "degree" : "BTech", "branch" : "CSE", "university" : "IIT, Kanpur"}
  print student_details                         #Printing whole dictionary

  print student_details["name"]                 #Robert James
  print student_details['age']                  #55
  print student_details['degree']               #BTech

  keys = student_details.keys()                 #creating list of keys   
  for key in keys:                              #iterating through created list of keys
      print key," ==> ",student_details[key]    #printing key and value referenced by it



  
 #Example - 3 (Pretty printing dictionary)

  import json       #importing json module

  student_details = {"name" : "Robert James", "age" : 55, "degree" : "BTech", "branch" : "CSE"}    #Defining a dictionary

  pretty_format_details = json.dumps(student_details,indent=4)  #dumps() function returns the string   
                                                                #representation of dictionary based on the keyworded arguments(if provided)
  print pretty_format_details



  
 #Example - 4 (Pretty printing dictionary in sorted keys order)

  import json       #importing json module

  student_details = {"name" : "Robert James", "age" : 55, "degree" : "BTech", "branch" : "CSE"}    #Defining a dictionary
  pretty_format_key_sorted_details = json.dumps(student_details,indent=4,sort_keys=True)
  print pretty_format_key_sorted_details

 """Designing a function that takes 1 dictionary as an argument & returns a list 2 items. 
    First item will be a tuple containing all the keys & second item will be a list of values."""

  def key_value_separator(details):
      return [tuple(details.keys()), details.values()]          

  my_details = {  "name": "Rishikesh Agrawani", \               #Dictionary spanned in 4 lines
                  "age" : 24, \
                  "branch" : "CSE" \
               }
     
  print key_value_separator(my_details)                         #[('age', 'name', 'branch'), [24, 'Rishikesh Agrawani', 'CSE']]
