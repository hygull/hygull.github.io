 Python - List


Tuple is defined as an mutable list of data items. We can modify the items of a list after its creation, and this is one of the important difference between tuple and list .

 #Interactive mode programming - #1

  >>> l = [1,2,3,4,5]
  >>>
  >>> print l
  [1, 2, 3, 4, 5]
  >>> 
  >>> type(l)
  <type 'list'>
  >>> 
  >>> l[0]
  1
  >>> l[4]
  5
  >>> l[4] = 67
  >>> 
  >>> print l
  [1, 2, 3, 4, 67]
  >>> 
  >>> l.append(69)                  #Apending item to list
  >>> 
  >>> print l
  [1, 2, 3, 4, 67, 69]
  >>> 
  >>> print l.remove(4)             #Removing item from list
  None
  >>> 
  >>> print l
  [1, 2, 3, 67, 69]
  >>> 
  >>> print l.remove(333)
  Traceback (most recent call last):
    File "", line 1, in 
  ValueError: list.remove(x): x not in list
  >>> 
  >>> print l.pop()                 #Popping out item from list
  69
  >>> 
  >>> print l
  [1, 2, 3, 67]
  >>> 
  >>> print l.pop()
  67
  >>> print l.pop()
  3
  >>> print l.pop()
  2
  >>> print l.pop()
  1
  >>> print l.pop()
  Traceback (most recent call last):
    File "", line 1, in 
  IndexError: pop from empty list
  >>>
